 # 💫 About Me:
Hi 👋, I'm Rohan Nigam, a motivated and detail-oriented Software Developer with a strong foundation in full-stack web development and problem-solving. I hold a Master of Computer Applications (MCA) from Manipal University Jaipur with a CGPA of 9.0 and a Bachelor of Computer Applications (BCA) with a CGPA of 9.0.<br><br>💻 I have hands-on experience working with **C, C#, Java, Python, and JavaScript**, and I am proficient in databases such as **MySQL, MongoDB, Oracle, and SQLite**. My frontend expertise includes **HTML5, CSS3, JavaScript, and React.js**, while on the backend I have built scalable applications using **Django and the MERN stack**.<br><br>🚀 I have worked as a Web Developer intern at Cosmo Info Solutions and as a MERN Stack Developer through Techkriti in collaboration with IIT Kanpur, where I developed real-time applications, improved performance, and collaborated in agile team environments. I have also built impactful projects like **MUJ Unite**, **D.N.E (Development Network for Education)**, and an **E-Library & E-Commerce platform**, focusing on user authentication, role-based access, messaging, and optimized user experience.<br><br>🧠 I follow **Object-Oriented Programming (OOP)** and **SOLID principles**, practice **Data Structures and Algorithms**, and have experience with **manual testing and Selenium**. Along with technical skills, I bring strong **analytical thinking, management, and time management abilities**.<br><br>🎯 I am passionate about learning new technologies, building real-world applications, and contributing to meaningful software projects as a **Full Stack / Software Developer**.<br>
